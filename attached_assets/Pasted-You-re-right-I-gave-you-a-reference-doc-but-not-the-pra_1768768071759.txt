## What Happens After Seeding

Once flows are in the database:

1. **Start a claim** → Set the peril type (water, hail_wind, fire, theft)
2. **Go to Workflow tab** → Click "Start Inspection"
3. **System auto-selects the matching flow** based on peril
4. **Walk through phases and movements** — this is the guided inspection
5. **Capture evidence** — photos get tagged with taxonomy prefixes
6. **Complete flow** → triggers scope generation

---

## The Bottom Line

**You can't upload flows through settings UI.** They need to be seeded into the database. Give Claude Code that prompt and it will create a seed script that inserts all 4 flows.

Then you can demo the full power: pick a claim, start the flow, walk through a guided inspection.
